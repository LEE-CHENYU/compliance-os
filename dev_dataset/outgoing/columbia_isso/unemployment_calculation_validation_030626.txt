PURPOSE

Validate the unemployment arithmetic using an actual unemployment-day method
and list all employment periods used in that calculation.

COUNTING METHOD USED IN THIS VALIDATION

Use actual unemployment days only:

- If a student becomes unemployed before the first job starts, count from the
  first day of authorized OPT unemployment through the day before the first
  job starts.
- If one job ends and another job starts later, count from the day after the
  earlier job ends through the day before the next job starts.
- Do not count the next job's start date as unemployment.

Examples used here:

- OPT start 01/23/2023 -> first employer start 04/21/2023 = 88 days
- Claudius ended 02/06/2025, so actual unemployment is 02/07/2025 - 02/19/2025 = 13 days
- CliniPulse ended 03/11/2025, so actual unemployment is 03/12/2025 - 03/16/2025 = 5 days
- If Tiger Cloud effectively ended 11/30/2024, actual unemployment before
  Claudius is 12/01/2024 - 12/15/2024 = 15 days
