--- RESOLUTION (03/10/2026, via thinkorswim live chat) ---

Root cause: LLC account *9239 is NOT thinkorswim-enabled.
  - Without thinkorswim enablement, the Trader API will NOT return:
    - Full account access / positions
    - Order entry capabilities
    - Order or transaction history
  - initialBalances still shows correct values (legacy data), but
    currentBalances and positions require thinkorswim enablement.

Fix: Enable thinkorswim for the LLC account via:
  schwab.com → Trade tab → Trading Platforms → "Learn how to enable thinkorswim"
  Takes overnight to process.

Once enabled, API will return full positions and correct currentBalances.
