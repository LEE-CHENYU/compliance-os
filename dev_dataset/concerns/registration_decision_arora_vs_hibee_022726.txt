2. WHAT MATTERS FOR REGISTRATION (and what doesn't)
================================================================================

  REGISTRATION IS MECHANICAL. The attorney:
    1. Enters employer info (BSGC LLC name, address, EIN)
    2. Enters beneficiary info (name, DOB, passport, country)
    3. Enters job info (title, SOC code, work location, wage level)
    4. Submits passport bio page copy
    5. Clicks submit on myUSCIS portal

  THIS TAKES ~15 MINUTES. No legal strategy. No documents. No analysis.

  WHAT MATTERS FOR REGISTRATION:
    ✓ Can they enter data correctly? → Both can.
    ✓ Can they select correct SOC code and wage level? → Both can.
    ✓ Can they meet the March 4-19 deadline? → Both confirmed.
    ✓ Do they already have our data? → Arora: YES. Hibee: PARTIAL.
    ✓ Is the fee reasonable? → Both comparable ($663 vs $636).

  WHAT DOES NOT MATTER FOR REGISTRATION:
    ✗ STEM OPT competence (irrelevant at registration stage)
    ✗ Modernization Rule depth (irrelevant — just data entry)
    ✗ Communication style (minimal interaction needed)
    ✗ Entity structure (Hibee vs law firm — irrelevant for data entry)
    ✗ Malpractice insurance (no legal judgment involved)
    ✗ Published case wins (no legal strategy involved)

4. THE WEIGHTED LOTTERY FACTOR — SOC CODE & WAGE LEVEL
================================================================================

  THIS IS THE ONE AREA WHERE REGISTRATION IS NOT PURELY MECHANICAL.

  FY2027 uses wage-based weighted selection:
    Level IV = 4x entries
    Level III = 3x entries
    Level II = 2x entries
    Level I = 1x entry

  The wage level is determined by:
    (a) The SOC code selected (which occupation)
    (b) The work location (which geographic area)
    (c) The offered wage (which prevailing wage level it falls into)

  For a "Financial Specialist" at BSGC LLC:
    - SOC code matters — different financial SOC codes have different
      prevailing wages for the same location
    - The offered wage must meet or exceed the prevailing wage for that
      SOC code in that location
    - A higher wage level = more lottery entries = better odds

  WHO IS BETTER AT THIS?

  ARORA: Has extensive H-1B practice. Selected SOC codes for 80+ LCA
  filings. Suggested "Financial Specialist" as job title during
  consultation. Has detailed website content on registration including
  SOC code selection.

  HIBEE: Claims 1,000+ H-1B cases. Presumably experienced with SOC
  code selection. But no specific evidence of their approach.

  EDGE: ARORA — he's already discussed the job title and likely has a
  SOC code in mind. He's also demonstrated understanding that the
  Modernization Rule allows non-specialty duties (business ownership)
  alongside specialty duties.
